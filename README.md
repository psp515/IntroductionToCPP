### About 

Repository contains solutions of C++ labolatories from C++ course on AGH UST.
Topics covered:
- operators overloading
- constructors
- templates
- lambda expressions
- inheritance and polymorphism in C++
- exceptions
- smart pointers
- STL: iterators, containers...
